Langkah Testing

Step 1: Login Admin
  1. Buka: http://localhost:5055/Auth/Login
  2. Username: admin
  3. Password: admin123
  4. Klik "Login"
Step 2: Akses Menu Karyawan
  1. Di sidebar admin, klik menu "Karyawan"
  2. Atau akses langsung: http://localhost:5055/Admin/Employees
  3. Verifikasi halaman terbuka dengan benar
Step 3: Tambah Karyawan Baru
  1. Klik tombol "Tambah Karyawan"
  2. Isi form dengan data berikut:
ID Karyawan EMP001
Nama Lengkap Siti Nurhaliza
Jabatan Kasir
No. HP 081234567890
Email siti@babyshop.com
Alamat Jl. Merdeka No. 123, Jakarta
Tanggal Masuk 01/01/2024
Status Aktif
Shift Pagi
Gaji 4500000
  1. Klik "Simpan"
  2. Verifikasi muncul pesan sukses
  3. Verifikasi karyawan muncul di tabel
Step 4: Edit Karyawan
  1. Klik tombol "Edit" (ikon pensil) pada karyawan yang baru ditambahkan
  2. Ubah beberapa data (contoh: ubah shift ke "Siang")
  3. Klik "Simpan Perubahan"
  4. Verifikasi data ter-update
Step 5: Lihat Detail Karyawan
  1. Klik tombol "Detail" (ikon mata) pada karyawan
  2. Verifikasi semua informasi tampil lengkap
  3. Cek perhitungan "Lama Bekerja" otomatis
  4. Verifikasi badge warna sesuai status dan shift
Step 6: Cek Statistik
  1. Kembali ke halaman Employees
  2. Scroll ke bawah untuk melihat 4 card statistik
  3. Verifikasi angka statistik sesuai dengan data
Step 7: Hapus Karyawan (Opsional)
  1. Klik tombol "Hapus" (ikon tempat sampah)
  2. Verifikasi muncul modal konfirmasi
  3. Klik "Hapus" untuk konfirmasi
  4. Verifikasi karyawan terhapus dari tabel

Test Validasi

  • Test ID Karyawan unique (coba tambah dengan ID yang sama)
  • Test validasi No. HP (coba input format salah)
  • Test validasi Email (coba input format salah)
  • Test field wajib (coba submit form kosong)
  • Test max length (coba input terlalu panjang)
  • Test date picker untuk Tanggal Masuk
  • Test dropdown Jabatan, Status, Shift
  • Test input Gaji (hanya angka)

Troubleshooting

Solusi:
  1. Pastikan sudah login sebagai admin
  2. Clear browser cache dan refresh
  3. Cek file _AdminLayout.cshtml sudah ter-update

Solusi:
  1. Stop aplikasi (Ctrl+C)
  2. Hapus file babyshop.db
  3. Jalankan ulang: dotnet run
  4. Database akan dibuat ulang dengan tabel Employee
