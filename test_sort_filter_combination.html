Verifikasi Kategori + Urutan Bekerja Bersamaan

üìã What to Test

Memastikan dropdown "Urutkan" berfungsi otomatis dan bekerja bersamaan dengan filter "Kategori".

  • Auto-submit saat pilih urutan (Default, Nama A-Z, Harga Terendah, Harga Tertinggi)
  • Kombinasi kategori + urutan bekerja dengan benar
  • Loading indicator muncul saat processing
  • Selected option ter-highlight dengan benar
  • URL parameters tersimpan dengan benar

Scenario 1: Sort Only (No Category Filter)

‚úì Expected:
- Halaman reload otomatis
- Hanya produk "Pakaian Bayi" yang muncul
- Produk diurutkan dari harga terendah
- URL: /Product?categoryId=X&sortBy=price_asc
Scenario 3: Category + Sort (Harga Tertinggi)
1. Pilih kategori "Mainan Bayi"
2. Tunggu produk ter-filter
3. Pilih "Harga Tertinggi" dari dropdown Urutkan
‚úì Expected:
- Halaman reload otomatis

- Produk diurutkan dari harga tertinggi
- URL: /Product?categoryId=X&sortBy=price_desc

1. Pilih "Harga Terendah"

Scenario 6: Reset After Multiple Filters
1. Apply kategori + sort + search
2. Klik tombol "Reset Filter"
‚úì Expected:
- Semua filter cleared
- Kategori = "Semua Kategori"
- Urutan = "Default"

- URL: /Product
Scenario 7: Back to Default Sort
1. Pilih "Harga Terendah"
2. Tunggu hasil
3. Pilih "Default" dari dropdown
‚úì Expected:
- Halaman reload otomatis
- Produk kembali ke urutan default (by ID)

üîç What to Check

  • Auto-Submit: Dropdown langsung submit saat berubah
  • Loading Indicator: Spinner pink muncul saat processing
  • Selected State: Opsi yang dipilih ter-highlight

  • Product Order: Produk benar-benar terurut sesuai pilihan
  • Combination: Kategori + Sort bekerja bersamaan
  • No Errors: Tidak ada error di console browser
